# CNN for MNIST — Vibe Coding Homework
 
A hands-on deep learning exercise where you build a 2D CNN for handwritten digit classification **entirely through an AI coding agent** (GitHub Copilot / Codex in VS Code), without writing code from scratch.
 
## Objectives
 
- Get a first experience of **vibe coding**: driving an AI agent with natural-language prompts to produce working research code.
- Build and iterate on a **2D CNN in PyTorch** for image classification.
- Learn to critically **verify and audit** agent-generated code (data splits, input formats, model choices).
- Compare CNN performance against **classical ML baselines**.
